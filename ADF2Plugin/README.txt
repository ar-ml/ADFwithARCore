Android fullscreen webview plugin written by Lubos

to compile:

JAR
download ADT: https://stackoverflow.com/questions/27418096/where-can-i-download-eclipse-android-bundle
download target api 20
set sdk path in local.properties (use)
install ant (easier on linux/osx)
run "ant clean" and then "ant jar"

SO
download and install android ndk
in this folder call ndk-build
